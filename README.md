# why-trip - backend

## 기술 스택
* 자세한 내용은 build.gradle 파일을 참고해주세요.
- Spring Boot 2.7.12
- jdk 17
- Gradle-7.6.1
- MariaDB (prod,dev) , H2 (local)
- Spring Security (with JWT token)
    - jwt dependency : io.jsonwebtoken
- [Spring Data JPA](https://docs.spring.io/spring-data/jpa/docs/current/reference/html/)
- [Query DSL](https://querydsl.com/)
- Swagger (spring openapi docs)
- docker-compose
